# РемСервис-Планировщик

Автоматизированная система планирования мобильных аварийно-ремонтных бригад.

## Описание

Система автоматизирует распределение заявок на обслуживание между мобильными ремонтными бригадами. Оптимизирует маршруты с учётом специализации бригад, приоритета заявок, географической близости и временных окон смен. Формирует маршрутные листы в формате PDF для выдачи бригадам.

## Быстрый старт

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

Наполнить тестовыми данными:

```bash
python seed.py          # бригады + 5 заявок на сегодня
python create_requests_april5.py  # 20 заявок на 2026-04-05
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend доступен на http://localhost:5173, API проксируется на порт 8000.

## Использование

1. **Создать заявки** -- перейти на "Заявки" > "Создать заявку", заполнить форму. Или запустить скрипт `create_requests_april5.py`.

2. **Запустить планирование** -- перейти на "Планирование", выбрать дату, нажать "Быстрый план" (жадный алгоритм) или "Оптимальный план" (OR-Tools с учётом смены).

3. **Просмотреть маршруты на карте** -- перейти на "Карта", выбрать дату. Маршруты бригад отображаются цветными линиями, гаражи -- маркерами. Фильтрация по бригаде через выпадающий список.

4. **Скачать маршрутные листы** -- перейти на "Маршрутные листы", выбрать дату. Раскрыть блок бригады для просмотра маршрута. Нажать "Скачать PDF" для печати.

5. **Статистика** -- на странице "Планирование" отображаются суммарные показатели: количество заявок, бригад, общее время работы, овертайм, расстояние переездов.

## Структура проекта

```
remservice-planner/
├── backend/
│   ├── main.py                  # Точка входа FastAPI
│   ├── config.py                # Настройки (pydantic-settings)
│   ├── database.py              # Async SQLAlchemy engine
│   ├── seed.py                  # Тестовые данные (бригады + заявки)
│   ├── create_requests_april5.py # 20 заявок на конкретную дату
│   ├── requirements.txt
│   ├── models/                  # SQLAlchemy модели
│   │   ├── brigade.py           # Brigade, BrigadeMember
│   │   ├── vehicle.py           # Vehicle
│   │   ├── request.py           # ServiceRequest
│   │   └── route.py             # DailyPlan, RoutePoint, RouteSegment
│   ├── schemas/                 # Pydantic v2 схемы
│   ├── routers/                 # API эндпоинты
│   │   ├── requests.py          # CRUD заявок
│   │   ├── brigades.py          # Просмотр бригад
│   │   ├── planning.py          # Планирование, статистика, геометрия
│   │   └── route_sheets.py      # Маршрутные листы + PDF
│   └── services/                # Бизнес-логика
│       ├── planning_service.py  # Жадный алгоритм
│       ├── or_tools_service.py  # OR-Tools VRPTW
│       ├── osrm_service.py      # OSRM маршруты + матрица
│       ├── geocoding_service.py # Nominatim геокодирование
│       └── pdf_service.py       # Генерация PDF
└── frontend/
    ├── package.json
    ├── vite.config.js
    └── src/
        ├── main.js
        ├── App.vue
        ├── api/index.js          # Axios API-клиент
        ├── router/index.js       # Маршруты
        ├── stores/               # Pinia stores
        ├── views/                # Страницы
        └── components/           # Переиспользуемые компоненты
```
